# 📊 Excel Data Analysis Project

## 📌 Descrição do Projeto

Este projeto consiste na utilização do **Microsoft Excel para manipulação, análise e tratamento de dados**, aplicando diversas funções e fórmulas avançadas para resolver problemas de análise de dados e simular cenários comuns encontrados em ambientes de negócios.

A planilha contém múltiplas abas com exercícios e análises que envolvem **estatística básica, classificação de dados, agregações, consultas entre tabelas e construção de uma base de vendas integrada**.

O objetivo do projeto é demonstrar habilidades em **Excel aplicado à análise de dados**, incluindo manipulação de grandes volumes de informação e uso de fórmulas para geração de insights.

---

# 🎯 Objetivos da Análise

As atividades realizadas na planilha incluem:

* Análise estatística básica de dados
* Classificação automática de registros
* Segmentação de dados em níveis ou categorias
* Contagem e agregação de registros
* Cruzamento de informações entre tabelas
* Construção de base de dados de vendas
* Aplicação de funções de busca e relacionamento

Essas práticas são comuns em tarefas de **Data Analysis, Business Intelligence e preparação de dados para dashboards**.

---

# 🧠 Principais Funções Utilizadas

Durante a construção das análises foram utilizadas diversas funções importantes do Excel:

### Funções Estatísticas

* `SUM`
* `MEDIAN`

Utilizadas para cálculo de **totais e medidas estatísticas básicas**.

---

### Funções Lógicas

* `IF`
* `AND`

Essas funções foram utilizadas para **classificar dados em categorias**, como por exemplo segmentação de clientes ou valores em diferentes níveis.

---

### Funções de Contagem

* `COUNTIF`
* `COUNTIFS`

Aplicadas para **contagem condicional de registros**, permitindo análises baseadas em critérios específicos.

---

### Funções de Agregação Condicional

* `SUMIF`
* `SUMIFS`

Essas funções permitem calcular **somatórios baseados em filtros**, sendo muito utilizadas em análises financeiras e relatórios de vendas.

---

### Funções de Busca e Relacionamento de Dados

* `VLOOKUP`
* `XLOOKUP`

Essas funções foram utilizadas para **relacionar dados entre diferentes tabelas**, simulando o funcionamento de **joins em bancos de dados**.

---

### Funções de Referência Avançada

* `INDEX`
* `MATCH`

Utilizadas para realizar buscas mais flexíveis entre tabelas e bases de dados.

# 📂 Estrutura da Planilha

A planilha contém diversas abas organizadas por exercícios ou etapas de análise:

### Q1

Aplicação de **cálculos estatísticos básicos** para análise de dados.

---

### Q2

Classificação de registros utilizando **funções lógicas (IF)** para segmentar dados em categorias.

---

### Q3

Criação de um sistema de **classificação por níveis**, utilizando múltiplas condições com `IF` e `AND`.

---

### Q4

Aplicação de **contagens condicionais** utilizando `COUNTIF` e `COUNTIFS`.

---

### Q5

Cálculo de **totais condicionais** utilizando `SUMIF` e `SUMIFS`.

---

### Q6

Análise de dados com **agregações e buscas entre colunas**, combinando contagens, somatórios e consultas.

---

### Q7

Aplicação de **buscas avançadas com XLOOKUP**, permitindo encontrar dados relacionados dentro da base.

---

### Q8

Construção de uma **base de dados simulando um ambiente de vendas**, composta por:

* cadastro de produtos
* cadastro de clientes
* base de vendas

Essas tabelas se relacionam entre si utilizando funções de busca.
